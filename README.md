# Primeiro jogo feito em Unity 3d

=======================================================================================

## Projeto Pong v_0.1.

A criação desse projeto se baseia em estudos ainda em andamento sobre a criação de jogos na ferramenta Unity 3d.



#### Sobre o projeto

Essa é a versão 0.1 do jogo, ainda há bugs e mais implementações para experiência do usuário.



**Aprendizagem**:

Utilizando cursos e tutoriais grátis da internet, esse projeto me ensinou as mecânicas básicas de criação de um jogo, como movimentação, sons, game design pensando no usuário, organização de arquivos, além da linguagem C# para programação de jogos.

Esse projeto foi proposto em um dos cursos para ser um exemplo prático de implementação da programação básica.



**Bugs, interfaces faltando e UX para ajuste:**

* Velocidade da bola: sua velocidade vai aumentando gradativamente até ficar sem controle e sair da tela do jogo;
* Falta de um menu e de tutoriais;
* Falta de sistema de pontuação;
* Tela de game over;
* Menu de pausa;
* Opções para modo janela ou tela inteira;
* Velocidade das raquetes: contra a inteligência artificial, a raquete da IA aumenta sua velocidade de acordo com a velocidade da bola.

Há essas entre outros vários ajustes para o jogo, lembrando que esse é um projeto para estudo, nada profissional, a intenção é ir melhorando e implementando mais mecânicas. 



## Como jogar

Extraia o arquivo "Executavel.rar", dentro da pasta, execute o arquivo "BetaPong.exe". **Não há necessidade de qualquer instalação, para "desinstalar", basta apagar os arquivos baixados!**

**Player 1** é a raquete esquerda, para controlar utilize as teclas "W" para controlar a raquete para cima e "S" para controlar a raquete para baixo.

**Player 2** é a raquete direita, para controlar utilize as setas ":arrow_up:" para controlar a raquete para cima e ":arrow_down:" para controlar a raquete para baixo.

**OBS:** a IA assume o player 2 automaticamente, para sair da IA, basta apertar qualquer tecla de controle do player 2, para voltar ao comando da IA, tecle *enter*.
